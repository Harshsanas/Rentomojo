## Getting Started with Create React App


  npx create-react-app Rentomojo

## For express app


  npm init -y


## Library used:- 

  ### 1. Nodemon
  
    
      npm i nodemon
    
  ### 2. mongoose
  
      npm i mongoose
    
  ### 3. MaterialUI
  
    
     npm i @material-ui/core
     npm i @material-ui/lab
    
  ### 4. Styled Components
  
    
     npm i styled-components
    
  ### 5. Axios
  
    
     npm i axios
